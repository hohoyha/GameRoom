이벤트 보내기 받기

  • socket.emit('이벤트명',{메세지}); => 이벤트 보내기
  • socket.on('이벤트명',function(data){ }); => 이벤트 받기

  • socket.broadcast.emit('이벤트명',{메세지}); => 나를 제외한 클라이언트들
  • io.sockets.emit('이벤트명',function(data){ }); => 모든 클라이언트들

Room 처리 (그룹)

  • socket.join(‘roon name’) => 방참가
  • socket.leave(‘roon name’) => 방나가기

  • io.sockets.in(‘roon name’).emit(‘event’,message) => 방안에 모든 클라이언트들
  • socket.broadcast.to(‘roon name’).emit(‘event’,message) => 나를 제외한 방안에 클라이언트들

  • io.sockets.manager.rooms => 방목록
  • io.sockets.clients(‘roon name’) => 방안에 클라이언트들
